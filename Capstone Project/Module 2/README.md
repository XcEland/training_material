# Capstone Stage: Module 2

## Focus

Apply advanced T-SQL query design and optimization concepts to the capstone dataset.

## Outcomes

- Use joins to connect staging, reference, and observation-style tables.
- Use subqueries for validation checks.
- Use CTEs to structure multi-step data quality logic.
- Use window functions for trends, rankings, or period-on-period comparisons.
- Review execution plans and statistics output.
- Add appropriate indexes and record before/after evidence.

## Notes

Keep the capstone schema separate from module lab schemas by using `cap`.
