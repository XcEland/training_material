# Module 5: Statistical Analysis and Data Visualization with Python

Module 5 turns SQL Server extracts into statistical analysis, visualizations, simple predictive models, and an analytical report.

## Lab

```text
Module 5/labs/statistical-analysis-data-visualization-python/
```

The lab covers:

- descriptive statistics, hypothesis testing, and correlation analysis with SciPy
- Matplotlib and Seaborn charts for analytical reporting
- Scikit-learn fundamentals for classification and model evaluation
- time series trend analysis and simple forecasting
- a final report that combines SQL extraction, Python analysis, and visualization

The lab uses `TrainingDB` and creates objects under the `m5` schema.
