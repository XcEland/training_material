# Module 3: Stored Procedures, Functions, and Database Automation

Module 3 moves from query writing into reusable database engineering patterns.

## Lab

```text
Module 3/labs/stored-procedures-functions-triggers-automation/
```

The lab covers:

- Stored procedure design, parameters, output values, and execution logging
- Scalar and table-valued user-defined functions
- Triggers for audit trails and business rule enforcement
- Secure dynamic SQL with `sp_executesql` and `QUOTENAME`
- Automated data validation with error handling and violation logging

The lab uses `TrainingDB` and creates objects under the `m3` schema.
